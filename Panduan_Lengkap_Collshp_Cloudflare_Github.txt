TAHAP 5 - CRAWLER (GraphQL API)
==================================================

Crawler menggunakan Collshp GraphQL API:
1. POST /api/v3/gql/graphql dengan query getLinkLists
2. Dapat semua shortlink + metadata (totalCount ~1100+)
3. Filter link s.shopee.co.id
4. Playwright resolve redirect ke URL Shopee asli
5. Ambil shopid, itemid, image
6. Dedup by shortlink (bukan shopid/itemid)
7. Auto-hapus produk yang tidak ada di Collshp
8. Filter non-women products (65+ keywords)
9. Retry 1x dengan 10s delay untuk timeout/connection error
